# 🚀 Automated Productivity Tracker

## 📌 Description

An automated productivity tracker designed to help users manage daily tasks, build habits, and improve efficiency. The application provides a clean and interactive interface to track progress and stay consistent.
This project also includes backend integration for handling data storage and server-side operations.

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Node.js
* Express.js
* MongoDB

## ✨ Features

* ✅ Add, update, and delete tasks
* ✅ Track daily habits and productivity
* ✅ User-friendly and responsive UI
* ✅ Organized dashboard for better tracking
* ✅ Backend API integration
* ✅ Persistent data storage using database

## ⚙️ Installation & Setup

1. Clone the repository:
   git clone https://github.com/Muskansoni6260/Automated-Productivity-Tracker.git

2. Navigate to the project folder:
   cd Automated-Productivity-Tracker

3. Setup backend:
   cd backend
   npm install

4. Run backend server:
   node server.js

5. Open `index.html` in your browser
